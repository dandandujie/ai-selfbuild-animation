# ai-selfbuild-animation

An animated single-page experiment about recursive self-improvement: when AI builds itself.

The page renders a living grid animation with switchable AI-brand inspired visual variants:

- Anthropic
- ChatGPT
- Gemini
- DeepSeek
- Kimi
- GLM
- MiniMax
- Grok

## Use

Open `recursive-self-improvement-animation.html` in a browser.

No build step or server is required.
